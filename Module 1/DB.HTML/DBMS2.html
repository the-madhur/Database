File-Based System

Earlier, data was stored using a File-Based System, such as:

  • Excel
  • Word Documents
  • Text Files

Problems with File-Based System

  • Data Redundancy – The same data is stored multiple times.
  • Data Inconsistency – Updates in one file may not reflect in others.
  • Difficult Data Access – Retrieving specific data is time-consuming.
  • Security Issues – Controlling access to sensitive data is challenging.

Database Management System (DBMS)

A Database Management System (DBMS) helps manage data efficiently by allowing users to store, retrieve, update, and process data in a structured way.

Why Use DBMS?

  • A DBMS helps identify use cases where databases are needed.
  • It acts as a layer between applications and the database, allowing smooth interaction.
  • It enables users to query, update, and process data efficiently.

Main Goal of DBMS

The main goal of a DBMS is to ensure smooth retrieval and storage of data, making data management efficient and secure.

How DBMS Works?

A DBMS:

  • Acts as an interface between applications and the stored database.
  • Handles operations like:
    • Adding new data
    • Updating existing data
    • Removing unnecessary data
    • Processing data for queries and transactions

DBMS Software

Some popular DBMS software include:

  • MySQL
  • PgSQL
  • Oracle SQL
  • SQLite

Using these DBMS software, we can add, delete, update, and remove data stored in a database.

SQL - Structured Query Language

All DBMS software understand a common language called SQL (Structured Query Language).

DBMS interacts with the stored database, and to interact with the DBMS, we need a programming language like SQL. Different DBMS software adapt SQL in slightly different ways.
